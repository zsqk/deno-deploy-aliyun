# deno deploy aliyun

deno 部署 阿里云版本

本工具的功能:

1. 编译 deno 项目.
2. 以自定义运行时的方式上传编译后的 deno 项目到阿里云函数计算.
3. 编译 deno 项目时总是强制更新缓存 (可能变动).
4. 支持使用私有依赖的 deno 项目. (通过 `buildEnv.DENO_AUTH_TOKENS` 达到)

## 局限

本工具的局限不支持 初始化项目. 初始化项目包括但不限于:

1. 创建服务.
2. 创建函数.
3. (为服务) 配置日志.
4. (为服务) 配置 VPC.
5. (为函数) 配置域名.

如何配置见 deno + 阿里云最佳实践.
